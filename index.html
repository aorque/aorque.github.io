The Problem

Joined a pre-launch brand with a marketing brief that required the throughput of a 4–5 person ops pod. No team to hire. No time to hire. Built the system instead.

The Stack

  • Mattermost layer (2 agents): Cadence (Marketing Coordinator) handles cross-workstream routing and the master marketing calendar. Rem (Events & Tour Ops) owns inbox triage, agent memory, and daily syncs. Human team interfaces here.
  • Cursor execution layer (5 agents): Ram (Chief of Staff) runs daily ops and EOD ledgers. Sparx (Idea Engine) scores and pitches ideas. Norm (KB steward) governs canon and detects drift. Dexter (Deep Research) handles vendor recon and comp intel. Connie (Convention Chair) owns isolated convention planning.
  • Adjacent agents: Eve (EOD synthesis), Scribe (KB ingestion). Plus Unit 0 — Hermes — for personal life operations.

How It Runs

Brain dump in. Agents structure. I call the shot. Ship. Write learnings back to the rules file. The system gets sharper every week because the canon gets tighter every week. Six weeks in: 69 active tasks running across 13 workstreams in Notion as source of truth.
